# ⛏️ Caps - Vein Mining (连锁挖矿)

**Caps** is a lightweight, modern vein mining mod designed for **Minecraft 1.21+ (NeoForge)**. It allows you to mine entire veins of ores or chop down whole trees instantly by holding a key, saving you time and effort while maintaining vanilla balance.

---

## ✨ Features (特性)

* **Simple Activation**: Just hold the **`~` (Grave Accent)** key while mining. No complex toggles!

* **Vanilla Balance**:
    * Consumes **Durability** correctly for every block broken.
    * Consumes **Hunger/Saturation** correctly.
    * Plays standard block breaking **Sounds and Particles**.
    * Supports **Experience (XP)** drops.
* **Visual Feedback**: (Client-side) Highlights the blocks that will be mined when you hold the key.

---

## 🎮 How to Use (如何使用)

1.  Install **NeoForge** for Minecraft 1.21+.
2.  Drop the `caps-x.x.x.jar` into your `mods` folder.
3.  In-game, hold the **`~` key** (the key usually above Tab) and break a block.
4.  All connected blocks of the same type will be mined!

*Note: You can change the keybinding in `Options -> Controls -> Key Binds`.*

## 🇨🇳 中文介绍 (Chinese Description)

**Caps** 是一个专为 **Minecraft 1.21+ (NeoForge)** 设计的轻量化**连锁挖矿**模组。它可以让你通过按住快捷键，一次性破坏整片矿脉或整棵树木，省时省力，同时保持原版游戏的平衡性。

### ✨ 核心功能
* **按住即用**：默认按住 **`~` 键** (波浪号键，位于Tab键上方) 挖掘即可触发连锁。

* **原版平衡性**：
    * 正确扣除工具**耐久度**（挖多少扣多少）。
    * 正确消耗玩家**饥饿度**。
    * 拥有完整的**方块破碎音效**、粒子效果和**经验掉落**。
* **可视化提示**：当按住按键时，会高亮显示即将被连锁破坏的方块。
